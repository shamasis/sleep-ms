# sleep-ms

## Installing

```terminal
npm install sleep-ms --save;
```

## API

```javascript
var sleepms = require('sleep-ms');

// sleep for one second
sleepms(1000);
```

## Using CLI

```terminal
# sleep for 1 second
sleepms 1000;
```

> If you intend to run this using CLI, it is recommended that you do a global install using 
> `npm install sleep-ms -g;`
